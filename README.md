# 🚗 Vehicle Rental System

A Java-based console application that allows users to manage vehicle rentals efficiently. The system supports adding vehicles, viewing available vehicles, renting vehicles, and returning them with proper tracking.

---

## 📌 Features

- Add new vehicles to the system
- View all available and rented vehicles
- Rent a vehicle by user selection
- Return rented vehicles
- Simple and interactive console-based UI
- Object-Oriented Programming (OOP) structure

---

## 🛠️ Tech Stack

- **Language:** Java
- **IDE:** IntelliJ IDEA / VS Code
- **Concepts Used:** OOP, Classes, Objects, Arrays/Collections, Basic File Structure
